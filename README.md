# Work Order Analysis Dashboard

# Overview
An interactive Power BI dashboard developed to analyze work order operations across multiple districts and service categories. The dashboard helps track order volume, revenue, waiting time, warranty claims, and service profitability.

# Tools Used
- Power BI
- Power Query
- DAX
- Excel

# Key Metrics
- Total Orders: 1,000
- Total Revenue: ₹237K
- Average Waiting Time: 28.02
- Rush Orders: 10.3%

# Dashboard Features
- District, Service, and Date filters
- Monthly order trend analysis
- District-wise revenue analysis
- Service-wise order distribution
- Warranty analysis
- Profit/Loss analysis by service

# Key Insights
- Northwest district recorded the highest number of orders.
- Assess service had the highest workload.
- Labour and Parts warranty claims remained below 10%.
- Deliver and Assess services generated positive profitability.
